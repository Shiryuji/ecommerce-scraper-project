# E-commerce Data Scraper Project

โปรเจกต์ฝึกฝนทักษะ Data Engineering: ดึงข้อมูลสินค้าจากเว็บ e-commerce จริง (BaNANA IT),
ทำความสะอาดข้อมูล และจัดเก็บเป็นตาราง

**สถานะ:** กำลังพัฒนา (Phase 1: Web Scraping)

## เทคโนโลยีที่ใช้
- Python 3.14
- requests, BeautifulSoup4 (web scraping)
- pandas (data processing)

## หมายเหตุ
โปรเจกต์นี้ทำเพื่อการเรียนรู้/พอร์ตโฟลิโอส่วนตัวเท่านั้น (educational/personal use)
ดึงข้อมูลในปริมาณจำกัด และเว้นระยะเวลาระหว่าง request เพื่อไม่ให้เป็นภาระต่อเซิร์ฟเวอร์ปลายทาง
